Summary Page

This project consists of creating a series of Python scripts to visualize the weather of 500+ cities across the world at varying distance from the equator.

To accomplish this, I utilized several Python libraries, OpenWeatherMap API and my knowledge of Python coding to create 4 plots that compare the Latitude of each city against 4 other variables (Temperature, Humidity, Cloudiness, Wind)
